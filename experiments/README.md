# Experiments

This directory stores prototypes, scratch files, and low-level request tests used during Grepolis script development.

These files are not packaged or documented as end-user tools like the projects under [`Scripts/`](../Scripts/README.md).

## Files

| File | Purpose |
|------|---------|
| `town-info-attack-request.js` | Manual request prototype for opening Grepolis attack data |
| `game-source.js` | Archived legacy bundle kept as a reverse-engineering reference |
| `timed-command-retry.js` | Prototype for timed attack or support retry logic |

Use these files as development references, not as production-ready userscripts.
