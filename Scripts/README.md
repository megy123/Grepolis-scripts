# Scripts

This directory contains the public-facing Grepolis projects in this repository.

Typical contents of a project folder:

- `README.md` with scope, status, and usage notes
- `src/` with userscript or source files
- `figures/` or `resources/` with screenshots and supporting assets

## Included Projects

| Folder | Purpose | Status |
|--------|---------|--------|
| `CSDetector` | Incoming attack analysis and slowest-unit estimation | Beta |
| `RebellionReport` | Rebellion report generation for alliance workflows | Work in progress |
| `EyeOfArtemis` | Attack timing helper concept | Concept |
| `Tormentor` | Fake-attack interval helper concept | Concept |
| `Vanish` | Dodge-timing helper concept | Concept |
