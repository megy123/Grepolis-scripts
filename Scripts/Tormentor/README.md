# Tormentor

Concept project for sending fake attacks in predefined intervals.

## Status

Concept stage. The idea is defined, but the project is not yet presented as a stable public script.

## Goal

Tormentor is intended to automate repetitive fake-attack patterns so they can be launched with more consistent spacing and less manual interaction.

## Planned Scope

- Interval-based fake attack dispatching
- Reduced manual clicking during repetitive pressure actions
- Grepolis-specific workflow support for harassment patterns

## Notes

This folder currently serves as a placeholder for a future public implementation.
