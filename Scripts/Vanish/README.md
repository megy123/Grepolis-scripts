# Vanish

Concept project for automated dodge timing of offensive units in Grepolis.

## Status

Concept stage. The repository currently tracks the project direction, not a finalized public release.

## Goal

Vanish is intended to help move offensive stacks out of danger before incoming attacks land, with a focus on timing-sensitive defensive play.

## Planned Scope

- Automated or semi-automated dodge timing
- Reduced reaction-time pressure during incoming attacks
- Workflow helpers around outgoing and returning unit movement

## Notes

This project should currently be treated as experimental documentation rather than a ready-to-install userscript.
