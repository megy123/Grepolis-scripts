# Eye of Artemis

Concept project for precise Grepolis attack timing.

## Status

Concept stage. This folder currently documents the idea and scope, not a finished userscript.

## Goal

The planned purpose of Eye of Artemis is to help with exact attack arrival timing against a target town, reducing manual retry loops and timing guesswork.

## Planned Scope

- Timing assistance for coordinated attacks
- Support for repeatable launch calculations
- Grepolis-specific workflow automation around arrival windows

## Notes

This project is not yet documented as a public installable script and should be considered experimental.
